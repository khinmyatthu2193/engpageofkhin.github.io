"Spoken Patterns"


1. It is time to + V1 (... ဖို့ အချိန်တန်ပြီ)
2.Don't forget to + V1 (... ဖို့ မမေ့နဲ့နော်)
3.Need to + V1 (... လုပ်ဖို့လိုတယ်)
4.About to + V1 (...တော့မလို့/ လုဆဲဆဲ)
5.Used to + V1 (အရင်တုန်းက...လေ့ရှိတယ်)
6.Have to + V1 (...ရအုံးမယ်)
7.Would like to + V1 (...ချင်တယ်)
8.Would like + Obj + to V1 (...ပေးစေချင်တယ်)
9.Be + ready + to V1 (...ဖို့ အသင့်ပဲ)
10.Be + dying + to V1 (အရမ်းပဲ...ချင်တယ်/ သေလုမတတ်ပါပဲ)
11.Intention is + to V1 (ရည်ရွယ်ချက်က.. ဖို့ပါပဲ)
12.Be + supposed + to V1 (...မယ်လို့ ယူဆရတယ်/ မျော်လင့်တယ်)
13.Have something + V1 (...စရာ တစ်ခုရှိတယ်)
14.There are reasons + to V1 (...ရတဲ့ အကြောင်းတော့ရှိမယ်)
15.There are reasons + to V1 (...ရတဲ့ အကြောင်းတော့ရှိမယ်)
16.Be + not willing + to V1 (...ချင်ပါဘူး/ ဆန္ဒမရှိဘူး)
17.Be + not eager+ to V1 (... စိတ်မထက်သန်မှုမရှိဘူး)
18.It is important + to V1 (...ဖို့ အရေးကြီးတယ်)
19.Pretended + not to V1 (....ချင်ဟန်ဆောင်တယ်)
20.have been waiting + to V1 (...ဖို့ စောင့်ဆိုင်းနေတာ)
21.It is hard for (Obj) + to V1 (....ဖို့ ခက်တယ်)
22.It will be rude + to V1 (...ရင် ရိုင်းရာကျလိမ့်မယ်)
23.Promised + not to V1 (...တော့ဘူးလို့ ကတိပြုထားတယ်)
24.It takes (an hour) + to V1 (...ဖို့/ ရတာ နာရီကြာတယ်)
25.It takes (an hour) + to V1 (...ဖို့/ ရတာ နာရီကြာတယ်)
26.S + don't/doesn't know + how to V1 (ဘယ်လို.. မလဲဆိုတာ.. မသိဘူး)
27 . S + don't/doesn't know + when to V1 ( ဘယ်တော့...မလဲဆိုတာ မသိဘူး)
28.don't/ doesn't know where to V1 (ဘယ်နေရာကို... မလဲဆို တာ မသိဘူး)
28.don't/ doesn't know where to V1 (ဘယ်နေရာကို... မလဲဆို တာ မသိဘူး)
29.Do you prefer + to V1...? (.....ဘယ်ဟာကို ပိုပြီးနှစ်သက်လဲ)
30.It is very easy + to V1 (..ဖို့/ရတာ အရမ်းလွယ်တယ်)
31.There is no need + to V1 (...စရာ မလိုပါဘူး)
32.Do you want me + to V1 (...စေချင်တာလား)
33.I feel it is my duty + to V1 (..ဖို့ ငါ့တာဝန်ပဲလို့ ခံယူတယ်)
34.Please dont hesitate + to V1 (...ဝန်လေးမနေနဲ့)
35.Do you know how/when/where to V1 (...လဲဆိုတာ မင်းသိလား)
36.It is not that I don't want to V1 (မ...ချင်လို့ မဟုတ်ပါဘူး)
37.Have you made up your mind + to V1....? (...ဖို့ ဆုံးဖြတ်လိုက်ပြီလား)
38.S + cant help + Ving (... ချင်တာ မနေနိုင်တော့ဘူး/ မတ တ်နိုင်တော့ဘူး)
39.I feel like + Ving (...ချင်စိတ်ပေါက်တယ်)
40.Would you mind + Ving..? (အပန်းမကြီးဘူးဆိုရင်...)
41.It is no use + Ving (...လို့တော့ ဘာမှ အကြောင်းထူးလာမှာမဟုတ်ဘူး)
42.S + be + accustomed to + Ving (...ရတာ ကျင့်သားရနေ ပီ/ နေသားကျနေပြီ)
43.S + don't/doesn't + stand a chance of + Ving ( အလား အလာမမြင်ဘူး/ ဖြစ်နိုင်ခြေမရှိဘူး)
44.What If.....? (....ရင် ဘယ့်နှယ် လုပ်မလဲ)
45.Would you please pass me + N (လှမ်းပေးပါလား)
46.It is too late + V1 (ဖို့... အတော်နောက်ကျနေပြီ)
47.How long does it take + to V1 (...ဖို့ ဘယ်လောက်ကြာမှာလဲ)
48.Don't fall + to V1 (မပျက်ကွက်စေနဲ့)
49.I'm afraid of missing + N. (လွတ်သွားမှာကိုစိုးရိမ်နေတယ်)
50.Do you mind if S + V + O? (...ရင် မင်းအနှောင့်အယှက်ဖြစ် မလား)
51.How long have you been + V-ing? (...နေတာ ဘယ်လောက်ကြာပြီလဲ)
52.Could you tell me the way to + Place? (သွားရမည့်လမ်းပါပြပေးနိုင်မလား)
